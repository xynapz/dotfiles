* Overview

This document describes my bash shell configuration using classic Unix tools.

** File Structure

| File           | Purpose                    | When Loaded       |
|----------------+----------------------------+-------------------|
| =.profile=     | Environment, PATH          | Login shells      |
| =.bash_profile= | Sources .profile          | Login shells      |
| =.bashrc=      | Interactive shell config   | All interactive   |

* Environment (.profile)

** Editor
#+begin_src bash
export EDITOR="emacs -nw"
export VISUAL="emacs -nw"
#+end_src

** Syntax Highlighting (less)

Uses *GNU Source-highlight* for syntax-colored output in =less=.

#+begin_src bash
export LESSOPEN="| /usr/bin/src-hilite-lesspipe.sh %s"
export LESS='-R -i -M -S -x4'
#+end_src

Usage:
#+begin_src bash
less app.py          # Syntax highlighted
less /var/log/syslog # Works on any file
#+end_src

** Colored Man Pages

Uses LESS_TERMCAP variables for colored man output:
- Headers: Cyan
- Links: Green
- Search: Yellow on blue

* Shell Options (.bashrc)

** History

| Setting           | Value               | Purpose                    |
|-------------------+---------------------+----------------------------|
| HISTSIZE          | 10000               | Commands in memory         |
| HISTFILESIZE      | 20000               | Commands in file           |
| HISTCONTROL       | ignoreboth:erasedups | No dupes, ignore spaces   |
| HISTIGNORE        | ls:cd:pwd:exit      | Skip trivial commands      |

** Navigation

| Option      | Effect                           |
|-------------+----------------------------------|
| autocd      | Type dir name to cd              |
| cdspell     | Fix minor typos in cd            |
| globstar    | =**= matches nested directories  |

* Aliases

** Safety
#+begin_src bash
alias cp='cp -i'      # Confirm overwrite
alias mv='mv -i'
alias rm='rm -i'
#+end_src

** Navigation
#+begin_src bash
alias ..='cd ..'
alias ...='cd ../..'
alias ....='cd ../../..'
#+end_src

** Listing
#+begin_src bash
alias ll='ls -lh'     # Long format, human sizes
alias la='ls -lAh'    # Include hidden
alias lt='ls -lhtr'   # Sort by time, newest last
#+end_src

** Git
#+begin_src bash
alias gs='git status -sb'
alias gl='git log --oneline -20'
alias gd='git diff'
alias ga='git add'
alias gc='git commit'
alias gp='git push'
alias gf='git fetch --all --prune'
#+end_src

* Functions

** mkcd - Create and enter directory
#+begin_src bash
mkcd myproject
# Creates myproject/ and cd into it
#+end_src

** extract - Universal archive extractor
#+begin_src bash
extract file.tar.gz   # Works with tar, zip, 7z, rar, etc.
#+end_src

** ccat - Cat with syntax highlighting
#+begin_src bash
ccat app.py           # Colored output via source-highlight
#+end_src

** ff/fdir - Find files/directories
#+begin_src bash
ff config             # Find files matching *config*
fdir node             # Find directories matching *node*
#+end_src

** up - Go up N directories
#+begin_src bash
up 3                  # cd ../../..
#+end_src

* Oh-My-Bash

** Theme
=pzq= - Minimal prompt with git info.

** Plugins

| Plugin             | Shortcut   | Description                  |
|--------------------+------------+------------------------------|
| git                | -          | Git completions and prompt   |
| sudo               | =Esc Esc=  | Prepend sudo to last command |
| bashmarks          | =s=, =g=, =l= | Save/go/list directories  |
| colored-man-pages  | -          | Colored man pages            |

** Bashmarks Usage
#+begin_src bash
s myproject           # Save current dir as 'myproject'
g myproject           # Go to saved bookmark
l                     # List all bookmarks
d myproject           # Delete bookmark
#+end_src

* Key Bindings

** Bash Defaults
| Key             | Action                    |
|-----------------+---------------------------|
| =Ctrl+R=        | Reverse history search    |
| =Ctrl+L=        | Clear screen              |
| =Ctrl+A/E=      | Start/end of line         |
| =Ctrl+U/K=      | Delete to start/end       |
| =Ctrl+W=        | Delete word backward      |
| =Alt+.=         | Last argument of prev cmd |

** Oh-My-Bash
| Key        | Action           |
|------------+------------------|
| =Esc Esc=  | Prepend sudo     |
