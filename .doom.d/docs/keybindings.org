#+TITLE: Doom Emacs Keybindings
#+AUTHOR: Angel Dhakal
#+STARTUP: overview
#+OPTIONS: toc:2

* Leader Key (SPC)

| Key           | Action                    |
|---------------+---------------------------|
| =SPC .=       | Find file                 |
| =SPC ,=       | Switch buffer (workspace) |
| =SPC <=       | Switch buffer (all)       |
| =SPC SPC=     | Find file in project      |
| =SPC `=       | Switch to last buffer     |
| =SPC /=       | Search project (ripgrep)  |
| =SPC *=       | Search project for symbol |

* Code (SPC c)

| Key         | Action                    |
|-------------+---------------------------|
| =SPC c a=   | LSP code actions          |
| =SPC c d=   | Jump to definition        |
| =SPC c D=   | Jump to references        |
| =SPC c f=   | Format buffer/region      |
| =SPC c i=   | Find implementations      |
| =SPC c k=   | Jump to documentation     |
| =SPC c r=   | LSP Rename                |
| =SPC c t=   | Find type definition      |
| =SPC c x=   | List errors/diagnostics   |
| =SPC c e=   | Evaluate buffer/region    |
| =SPC c o=   | LSP Organize imports      |
| =SPC c j=   | Jump to symbol (workspace)|
| =SPC c S=   | LSP Symbols tree          |

* Go-to Keys (Normal Mode)

| Key   | Action                    |
|-------+---------------------------|
| =gd=  | Go to definition          |
| =gD=  | Find references           |
| =gi=  | Evil insert resume *      |
| =gr=  | Eval region (REPL)        |
| =gQ=  | Format region             |
| =K=   | Lookup documentation      |

/Note: =gi= is Evil's insert-resume, not implementations. Use =SPC c i= for implementations./

* Buffer (SPC b)

| Key       | Action                    |
|-----------+---------------------------|
| =SPC b b= | Switch buffer (workspace) |
| =SPC b B= | Switch buffer (all)       |
| =SPC b d= | Kill buffer               |
| =SPC b k= | Kill buffer               |
| =SPC b n= | Next buffer               |
| =SPC b p= | Previous buffer           |
| =SPC b s= | Save buffer               |
| =SPC b r= | Revert buffer             |

* File (SPC f)

| Key       | Action                    |
|-----------+---------------------------|
| =SPC f f= | Find file                 |
| =SPC f r= | Recent files              |
| =SPC f s= | Save file                 |
| =SPC f S= | Save as                   |
| =SPC f p= | Find file in config       |
| =SPC f R= | Rename/move file          |
| =SPC f D= | Delete file               |
| =SPC f y= | Yank file path            |

* Window (SPC w)

| Key       | Action                    |
|-----------+---------------------------|
| =SPC w v= | Split vertical            |
| =SPC w s= | Split horizontal          |
| =SPC w c= | Close window              |
| =SPC w w= | Other window              |
| =SPC w m= | Maximize window           |
| =C-w h/j/k/l= | Navigate windows      |

* Project (SPC p)

Uses Projectile.

| Key       | Action                    |
|-----------+---------------------------|
| =SPC p p= | Switch project            |
| =SPC p f= | Find file in project      |
| =SPC p s= | Search in project         |
| =SPC p a= | Add project               |

* Git (SPC g)

| Key       | Action                    |
|-----------+---------------------------|
| =SPC g g= | Magit status              |
| =SPC g G= | Magit status here         |
| =SPC g b= | Magit branch checkout     |
| =SPC g B= | Magit blame               |
| =SPC g /= | Magit dispatch            |
| =SPC g l= | Log (vc-gutter)           |
| =SPC g ]= | Next hunk                 |
| =SPC g [= | Previous hunk             |

* Search (SPC s)

| Key       | Action                    |
|-----------+---------------------------|
| =SPC s s= | Search buffer             |
| =SPC s p= | Search project            |
| =SPC s i= | Jump to symbol (imenu)    |

* Org Mode

| Key       | Action                    |
|-----------+---------------------------|
| =SPC o a= | Org agenda                |
| =SPC o c= | Org capture               |
| =SPC m l= | Store link (in org)       |

* Workspace (SPC TAB)

| Key           | Action                |
|---------------+-----------------------|
| =SPC TAB TAB= | Display workspaces    |
| =SPC TAB n=   | New workspace         |
| =SPC TAB d=   | Kill workspace        |
| =SPC TAB ]=   | Next workspace        |
| =SPC TAB [=   | Previous workspace    |
| =M-1= to =M-9=| Switch to workspace N |

* Help (SPC h)

| Key       | Action                    |
|-----------+---------------------------|
| =SPC h b b= | Browse keybindings      |
| =SPC h k=   | Describe key            |
| =SPC h f=   | Describe function       |
| =SPC h v=   | Describe variable       |
| =SPC h t=   | Evil tutor              |
